# Cafe-Manger
A web application for cafe inventory tracking, sales management, analytics and AI-assisted business insights
A modern web application for cafe owners to:

- Track inventory
- Record daily sales
- Monitor stock levels
- Visualize business analytics
- Receive AI-powered insights

## Tech Stack

- Python
- Streamlit
- PostgreSQL
- Pandas
- Plotly
- OpenAI API (planned)

## Features

- Inventory management
- Sales tracking
- Interactive dashboard
- CSV import
- AI business assistant (coming soon)
